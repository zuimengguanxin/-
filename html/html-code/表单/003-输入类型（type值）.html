7.输入类型：number 用于应该包含数字值的输入字段。

您能够对数字做出限制。

根据浏览器支持，限制可应用到输入字段。

实例

这里列出了一些常用的输入限制（其中一些是 HTML5 中新增的）：

属性 描述

disabled 规定输入字段应该被禁用。

max 规定输入字段的最大值。

maxlength 规定输入字段的最大字符数。

min 规定输入字段的最小值。

pattern 规定通过其检查输入值的正则表达式。

readonly 规定输入字段为只读（无法修改）。

required 规定输入字段是必需的（必需填写）。

size 规定输入字段的宽度（以字符计）。

step 规定输入字段的合法数字间隔。

value 规定输入字段的默认值。
